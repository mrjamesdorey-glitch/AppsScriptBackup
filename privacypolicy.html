Google Apps Script Backup Chrome Extension - Privacy Policy

Introduction

Google Apps Script Backup is a Chrome extension designed to help you securely back up your Google Apps Script projects as downloadable ZIP files. We are committed to protecting your privacy and ensuring transparency about how this extension handles your data.

Non-Affiliation with Google

Google Apps Script Backup is not a Google product and is not affiliated with or endorsed by Google or the Google Apps Script platform. All interactions with the Google Apps Script editor (https://script.google.com/) are performed via official Google APIs. The extension allows you to download and email your project files, with all processing done locally on your device.

Data Collection and Usage

The extension does not collect, store, or transmit any personal data beyond what is required for authentication and backup functionality. The extension operates primarily on your device and uses the following:

  • Local & Synced Storage: The extension uses Chrome’s chrome.storage.sync API to store your backup preferences (e.g., download option, email self/other, custom name) locally and synced across your signed-in devices. This data is not sent to any external servers.
  • Google OAuth2: The extension uses Google’s OAuth2 to authenticate and access the Apps Script and Gmail APIs. Your Google email is fetched once during login for display and optional email delivery, and is stored only in memory.
  • Project Data: When backing up, the extension fetches your project files via the official Apps Script API and creates a ZIP file locally. No project data is stored or transmitted beyond the immediate backup process.
  • Email Delivery: If enabled, the extension uses the Gmail API to send the ZIP file to your Google account or a custom email address. The email is generated and sent directly via Google’s servers.

Permissions

The extension requires the following permissions to function:

  • identity: For Google OAuth2 authentication to access Apps Script and Gmail APIs.
  • storage: To save and sync your backup preferences across devices.
  • activeTab, scripting, tabs: To detect the current project in the Apps Script editor.
  • Host Permissions: To access Google APIs for project fetching, user info, and email sending.

Third-Party Services

The extension does not use any third-party services, analytics, or tracking tools. All functionality is self-contained within the extension and Google’s official APIs.

Data Security

Your project files, settings, and email addresses are stored locally via Chrome’s storage API and are as secure as your device. Authentication tokens are managed securely by Chrome’s identity API and revoked on logout. We do not have access to this data, and it is not shared with any external parties beyond Google’s APIs for backup and email delivery.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time to reflect changes in the extension’s functionality or legal requirements. The latest version will always be available at this URL. This policy was last updated on November 12, 2025.
