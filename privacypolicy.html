Privacy Policy

Introduction

The Google Apps Script Backup extension is designed to help you backup your Google Apps Script projects as downloadable ZIP files. We are committed to protecting your privacy and ensuring transparency about how this extension handles your data.

Non-Affiliation with Google

The extension is not a Google product and is not affiliated with or endorsed by Google or the Google Apps Script platform. All modifications to the Google Apps Script website (https://script.google.com/) are cosmetic, applied locally on your device, and are fully reversible by uninstalling the extension or removing your settings. The extension allows you to backup your projects by fetching content via the Google Apps Script API and creating local ZIP files, while preserving the content you want to see.

Data Collection and Usage

The extension does not collect, store, or transmit any personal data beyond what is required for authentication and backup functionality. The extension operates entirely on your device and uses the following:

  • Local Storage: The extension uses Chrome’s chrome.storage.sync API to store your backup preferences (e.g., download options, email addresses, custom names) locally on your device and synced across devices. This data is not sent to any external servers.
  • Google OAuth2: The extension uses Google’s OAuth2 for authentication to access the Apps Script API. Your Google email is fetched once during login for email delivery and stored in memory only.
  • Project Data: When backing up, the extension fetches your project files via the Apps Script API and creates a local ZIP file. No project data is stored or transmitted beyond the immediate backup process.
  • Email Delivery: If you choose to email backups, the extension uses the Gmail API to send the ZIP file to the specified email addresses (your Google account or a custom email). The email content is generated locally and sent directly via Google's servers.

Permissions

The extension requires the following permissions to function:

  • identity: For Google OAuth2 authentication to access Apps Script and Gmail APIs.
  • storage: To save your backup preferences locally and synced across devices.
  • activeTab, scripting, tabs: To inject the content script into script.google.com for project detection.
  • Host Permissions: To access Google APIs for project fetching, email sending, and user info.

Third-Party Services

The extension does not use any third-party services, analytics, or tracking tools. All functionality is self-contained within the extension and Google's APIs.

Data Security

Your project files, settings, and email addresses are stored locally via Chrome's storage API and are as secure as your device. Authentication tokens are cached by Chrome's identity API and revoked on logout. We do not have access to this data, and it is not shared with any external parties beyond Google's APIs for backup and email delivery.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time to reflect changes in the extension's functionality or legal requirements. The latest version will always be available at this URL. This policy was last updated on November 12, 2025.
