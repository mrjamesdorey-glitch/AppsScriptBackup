Changelog

Version 1.0

November 12, 2025

  • Initial release of Google Apps Script Backup.
  • One-click backup of Apps Script projects as ZIP files.
  • Download to device or email to self/other address.
  • Custom backup name and timestamped filenames.
  • Secure Google OAuth2 sign-in with silent re-auth.
  • Auto-detects current project in script.google.com.
  • Settings sync across devices via chrome.storage.sync.
  • Clean, Google-inspired UI with smooth transitions.
